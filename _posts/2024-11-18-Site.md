# 유용한 사이트

## 디자인 관련

### 1. [Dribbble](https://dribbble.com/)
- **설명**: 디자이너들이 자신의 작업을 공유하고 피드백을 받을 수 있는 플랫폼. 다양한 디자인 영감을 얻을 수 있는 곳.

### 2. [Httpster](https://httpster.net/)
- **설명**: 최신 웹 디자인 트렌드를 소개하는 사이트. 다양한 웹사이트의 디자인을 탐색하고 영감을 받을 수 있음.

### 3. [Khroma](https://www.khroma.co/)
- **설명**: 색상 조합 추천 도구. 사용자가 선호하는 색상을 기반으로 다양한 색상 조합을 생성해줌.

### 4. [Manypixels](https://www.manypixels.co/gallery)
- **설명**: 무료 일러스트와 벡터 이미지를 제공하는 사이트. 다양한 주제의 일러스트를 다운로드할 수 있음.

### 5. [Storyset](https://storyset.com/)
- **설명**: 애니메이션된 일러스트를 제공하는 플랫폼. 다양한 스타일의 일러스트를 사용하여 프로젝트에 생동감을 더할 수 있음.

### 6. [WeTransfer](https://wetransfer.com/)
- **설명**: 대용량 파일 전송 서비스. 간편하게 파일을 업로드하고 링크를 통해 다른 사람과 공유할 수 있음.

### 7. [I Love PDF](https://www.ilovepdf.com/ko)
- **설명**: PDF 파일을 편집하고 변환할 수 있는 온라인 도구. PDF 파일을 쉽게 관리할 수 있는 다양한 기능을 제공.

## AI Code Editor

### 1. [GitHub Copilot](https://github.com/github-copilot/)
- **특징**: AI 기반 코드 자동 완성 도구로, 다양한 프로그래밍 언어를 지원.
- **장점**:
  - 코드 생산성 향상.
  - 다양한 언어 지원(C, Python, Java 등).
  - Visual Studio와 통합되어 사용 가능.
- **단점**:
  - 복잡한 프로젝트에서는 한계점이 있을 수 있음.
  - 일부 언어에서는 완성도가 낮음.
- **지원 언어**: JavaScript, Python, TypeScript, Ruby, Go, C#, C++, Java 등
- **가격**: 월 $10 또는 연간 $100 (개인 사용자 기준)

### 2. [Cursor](https://www.cursor.com/)

### AI 엔진 비교: GPT, Copilot, Claude

| AI 엔진 | 특징 | 장점 | 단점 |
|--------|------|------|------|
| **GPT**   | - 자연어 처리(NLP)에서 뛰어난 성능을 보임. <br> - 다양한 언어 모델 제공. | - 방대한 데이터로 훈련되어 높은 정확도와 유창성. <br> - API 통합 용이. | - 특정 주제에 대한 깊이 있는 지식 부족. <br> - 비논리적 응답 가능성. |
| **Copilot** | - 코드 작성 및 보조에 최적화된 AI. <br> - GitHub와 통합되어 사용.   | - 개발자 생산성 향상. <br> - 실시간 코드 제안 및 자동 완성. | - 특정 언어 및 프레임워크에 대한 제한적 지원. <br> - 코드 품질이 항상 보장되지 않음. |
| **Claude** | - 안전성과 윤리를 중시하는 AI 모델. <br> - 대화형 AI에 최적화.     | - 사용자 친화적인 응답 생성. <br> - 안전성 및 윤리적 고려 강화. | - 상대적으로 덜 알려진 모델로 특정 분야에서 성능 제한. <br> - API 접근성 낮을 수 있음. |

#### 결론
각 AI 엔진은 특정 용도와 요구에 따라 장단점이 다릅니다. GPT는 일반적인 텍스트 생성 및 대화형 AI에 강점을 보이며, Copilot은 개발자에게 유용한 코드 작성 도구로, Claude는 안전성과 윤리를 중시하는 사용자 친화적인 접근을 제공합니다. 필요에 따라 적절한 엔진을 선택하는 것이 중요합니다.

## AI 서비스

### 1. [ChatGPT](https://chatgpt.com/)
- **특징**: OpenAI의 언어 모델 기반 대화형 AI. 문장 생성, 질문 응답, 코드 작성, 콘텐츠 생성 등 다양한 작업 수행 가능.
- **장점**:
  - 자연스러운 대화와 고품질 텍스트 생성.
  - 다양한 주제에 대한 폭넓은 이해.
  - 커스터마이징 가능한 API 제공.
- **단점**:
  - 실시간 정보 업데이트 부족(기본적으로 2024년 이전 데이터 기준).
  - 긴 맥락 유지가 어려울 수 있음.

### 2. [Perplexity](https://www.perplexity.ai/)
- **특징**: 정보 검색 특화형 AI, 질문에 대해 간결하고 정확한 답변 제공.
- **장점**:
  - 실시간 웹 검색 기능 지원.
  - 간결한 정보 요약과 출처 제공.
  - 특정 질문에 대한 명확한 답변에 강점.
- **단점**:
  - 복잡한 작업이나 창의적 응답 생성에는 제한적.
  - 웹 의존도가 높아 네트워크 환경에 따라 성능이 달라질 수 있음.

### 3. [GenSpark](https://www.genspark.ai/)
- **특징**: AI 기반 생성형 플랫폼. 주로 텍스트, 이미지 생성 작업에 사용.
- **장점**:
  - 다양한 콘텐츠 생성 도구 통합.
  - 사용자 친화적 인터페이스.
  - 비교적 빠른 결과 생성.
- **단점**:
  - 전문성 높은 작업에서는 제한적일 수 있음.
  - 일부 사용자 맞춤화 기능 부족.

### 4. [Copilot](https://copilot.microsoft.com/)
- **특징**: Microsoft의 코드 보조 AI. 주로 개발자를 지원하며 코드 자동 완성 및 오류 수정 제안.
- **장점**:
  - 코드 생산성 향상.
  - 다중 언어 지원(C, Python, Java 등).
  - Visual Studio와 통합.
- **단점**:
  - 복잡한 프로젝트에서는 한계점이 있을 수 있음.
  - 일부 언어에서는 완성도가 낮음.

### 5. [NotebookLM](https://notebooklm.google)
- **특징**: Google이 제공하는 AI 기반 노트북 도구. 문서 기반 정보 요약과 분석 지원.
- **장점**:
  - 특정 문서와 연계된 질문 응답.
  - 연구 및 요약 작업에 최적화.
  - Google 문서 및 드라이브와 통합.
- **단점**:
  - 제한된 데이터 지원(외부 파일 분석 제한).
  - 실시간 웹 데이터 활용 부족.

### 6. [Github Copilot](https://github.com/github-copilot/)
- **특징**: GitHub과 OpenAI가 협력하여 만든 개발 보조 AI.
- **장점**:
  - 코드 자동 완성과 버그 수정 지원.
  - 코드 리뷰 및 리팩토링 도움.
  - 개발 효율성 극대화.
- **단점**:
  - 높은 가격.

### 7. [Adot.ai](https://adot.ai/)
